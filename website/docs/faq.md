# FAQ

## How frequently is gas data updated?
EthGasStation updates network fee metrics continuously as new chain data is ingested.

## Which networks are covered?
This documentation focuses on Ethereum mainnet gas metrics and alerts.

## Can I manage alerts through AI tools?
Yes. Use AI Connect (MCP) to create, list, pause, and delete alerts when your role has permission.

## What should I do if API calls start failing?
Check token validity, inspect HTTP status codes, and log `request_id` values from error responses before contacting support.

## How do I reduce alert noise?
Tune both threshold and cooldown settings. Start with a 30-minute cooldown and adjust using observed trigger frequency.
