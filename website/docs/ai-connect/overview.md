# AI Connect (MCP)

AI Connect exposes EthGasStation capabilities over Model Context Protocol (MCP) so AI clients can retrieve gas data and execute alert operations on your behalf.

## What you can do

- Request current gas conditions and lane-specific fee guidance
- Compare live values against recent ranges
- Create, list, pause, and delete alerts from chat

## Security model

- User authentication is handled through OAuth with explicit consent
- Tokens are scoped to allowed actions
- Access can be revoked at any time from EthGasStation account settings
