# Usage

After setup, AI Connect can read gas data and manage alerts using your account permissions.

## Common workflows

### Check market conditions

```text
What’s gas right now, and should I send or wait?
```

### Compare against recent context

```text
How does the current Average lane compare to the 24h range?
```

### Create an alert

```text
Create an alert when Average gas is at or below 18 gwei and notify me by email.
```

### Review active alerts

```text
List my active gas alerts with thresholds and cooldowns.
```

### Disable a recent alert

```text
Disable my most recent alert.
```

## Operational guidance

- use explicit units such as `gwei`
- prefer app lane names `Slow`, `Average`, and `Fast`
- ask for confirmation before write actions in shared/team environments
- re-authenticate if your client reports expired consent
